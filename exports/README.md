# exports/ — Standalone Modules for External Use

Self-contained packages that can be used independently of the toolkit.

## Contents

| Export | Purpose |
|--------|---------|
| deal-analyzer/ | Standalone deal analysis module (no toolkit dependencies) |

## What belongs here

- Modules that external consumers can use without installing the full toolkit
- Each export should be fully self-contained with its own dependencies

## What does NOT belong here

- Internal skills (goes in skills/)
- Shared libraries (goes in lib/)
